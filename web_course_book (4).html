HTML Негиздери

HTML (HyperText Markup Language) — веб-баракчаларды түзүүдө колдонулган белгилөө тили. Ал 1991-жылы Тим Бернерс-Ли тарабынан иштелип чыккан.

HTML деген эмне?

HTML — веб-баракчанын структурасын жана мазмунун аныктаган тил. Ал программалоо тили эмес, белгилөө тили (markup language). Браузер HTML кодун окуп, колдонуучуга баракчаны көрсөтөт.

🏗️

Структура

HTML баракчанын "скелети" — аталыш, абзац, сүрөт, шилтеме.

🌐

Браузер

Chrome, Firefox, Safari HTML кодун окуп, сизге баракча катары көрсөтөт.

📁

Кеңейтүү

HTML файлдары .html же .htm кеңейтүүсүн колдонот.

HTML документинин структурасы

Ар бир HTML файлы стандарттык структурага ээ болушу керек:

HTML 
<!-- DOCTYPE — HTML5 экенин билдирет -->
<!DOCTYPE html>
<html lang="ky">

  <head>
    <!-- Мета-маалымат (колдонуучу көрбөйт) -->
    <meta charset="UTF-8">
    <meta name="viewport"
          content="width=device-width">
    <title>Менин баракчам</title>
  </head>

  <body>
    <!-- Колдонуучу көрүүчү мазмун -->
    <h1>Салам, дүйнө!</h1>
    <p>Бул менин биринчи баракчам.</p>
  </body>

</html>
💡 Маанилүү <head> ичиндегилер браузер өтмөгүндө жана SEO үчүн. <body> ичиндегилерди гана колдонуучу экранда көрөт.

Семантикалык HTML

Семантикалык теглер мазмунунун маанисин билдирет. Алар SEO жана жеткиликтүүлүк үчүн маанилүү.

Семантикалык теглер

HTML 
<body>

  <header>     <!-- Баракчанын башы -->
    <nav></nav>  <!-- Навигация -->
  </header>

  <main>       <!-- Негизги мазмун -->
    <article>   <!-- Өз алдынча мазмун -->

CSS Негиздери

CSS (Cascading Style Sheets) — HTML элементтерине стиль берет. Түс, шрифт, өлчөм, жайгаштыруу — баардыгы CSS аркылуу.

CSS кантип иштейт?

CSS 
/* Синтаксис: сайлагыч { касиет: маани; } */

h1 {
  color: #2c1a0a;
  font-size: 32px;
  text-align: center;
  margin-bottom: 20px;
}

p {
  color: #555;
  font-size: 16px;
  line-height: 1.7;
}

body {
  background-color: #f5ede0;
  font-family: 'Georgia', serif;
  margin: 0;
}
💡 Куту моделі (Box Model) CSS'те ар бир элемент — бул куту. Анын ичинде контент, padding (ички бошлук), border (чек ара) жана margin (тышкы бошлук) бар.
CSS 
.кутуча {
  padding: 20px;          /* Ички бошлук */
  border: 2px solid #333; /* Чек ара */
  border-radius: 8px;     /* Тегерек бурч */
  margin: 16px auto;      /* Тышкы + борборго */
  width: 300px;
  box-sizing: border-box; /* Padding өлчөмгө кирсин */
  background: linear-gradient(135deg, #8B4513, #5C2E00);
  color: white;
}

Flexbox жана Grid

Flexbox жана Grid — заманбап CSS жайгаштыруу системалары. Алар менен татаал макеттерди оңой жасоого болот.

Flexbox

CSS 
.контейнер {
  display: flex;
  flex-direction: row;           /* row | column */
  justify-content: space-between; /* Горизонталдык */
  align-items: center;            /* Вертикалдык */
  flex-wrap: wrap;
  gap: 16px;
}

.бала { flex: 1; } /* Бош орунду тең бөлүш */

CSS Grid

CSS 
.тор {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

/* Адаптивдүү тор */
.адаптив {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(200px, 1fr));
  gap: 16px;
}

/* Элемент бир нече мамычаны ал */
.кең { grid-column: 1 / -1; }

Анимациялар & Media Query

CSS анимациялары веб-баракчаны жандуу кылат. Media queries жооп берүүчү дизайн үчүн зарыл.

Анимациялар

CSS 
/* Transition — жайлап өтүш */
.баскыч {
  background: #6b3aa8;
  transition: all 0.3s ease;
}
.баскыч:hover {
  background: #4a2080;
  transform: translateY(-3px);
  box-shadow: 0 8px 24px rgba(0,0,0,0.2);
}

/* @keyframes анимация */
@keyframes жарк-эт {
  0%   { opacity: 0; transform: translateY(20px); }
  100% { opacity: 1; transform: translateY(0); }
}

.пайда-болуу {
  animation: жарк-эт 0.5s ease forwards;
}

Media Queries

CSS 
/* Desktop (чоң экран) */
.тор { grid-template-columns: repeat(3, 1fr); }

/* Планшет */
@media (max-width: 768px) {
  .тор { grid-template-columns: repeat(2, 1fr); }
}

/* Телефон */
@media (max-width: 480px) {
  .тор { grid-template-columns: 1fr; }
  h1   { font-size: 24px; }
}

JavaScript Негиздери

JavaScript — веб-баракчаны интерактивдүү кылган программалоо тили. Логика, эсептөөлөр, маалымат иштетүү.

Өзгөрмөлөр жана маалымат түрлөрү

  console.log("А — Мыкты!");

  console.log("Б — Жакшы");
} else {
  console.log("Аракет кылыңыз");
}

// Тернар оператор
let статус = упай >= 50 ? "Өттү" : "Өтпөдү";

Функциялар жана Массивдер

Функциялар — коддун кайра колдонула турган блоктору. Массивдер — маалыматтардын тизмеси.

Функциялар

JS 
// Классикалык функция
function салам(ат) {
  return `Салам, ${ат}!`;
}

// Жебе функциясы
const квадрат = (сан) => сан * сан;

// Демейки параметр
function кош(ат = "мейман") {
  return `Кош келдиңиз, ${ат}!`;
}

// Рекурсия
function факториал(н) {
  if (н <= 1) return 1;
  return н * факториал(н - 1);
}
факториал(5); // → 120

DOM менен иштөө

DOM (Document Object Model) — JavaScript'ке HTML элементтерин динамикалык түрдө өзгөртүүгө мүмкүндүк берет.

Элементтерди табуу жана өзгөртүү

JS 
// Табуу
const баш = document.getElementById("аталыш");

const баары = document.querySelectorAll("li");

// Мазмун өзгөртүү
бир.textContent = "Жаңы текст";
бир.innerHTML = "<b>Жоон</b> текст";

// Стиль өзгөртүү

бир.style.display = "none";

// Класс кошуу/алуу

// Жаңы элемент жаратуу

// Жок кылуу

Fetch API + async/await

JS 
async function маалыматАлуу() {

    console.log(маалымат);
  } catch (ката) {
    console.error("Ката:", ката);
  }
}
